Place your hero background video here as: hero.mp4

Recommended specs:
- Format: MP4 (H.264), muted-friendly (no important audio, video is muted by default)
- Resolution: 1920x1080 (or 1280x720 for a smaller file size)
- Length: 10-20 seconds, looping seamlessly
- File size: keep it as small as possible (a few MB) since it autoplays on page load

The template at src/main/resources/templates/home.html references:
  /videos/hero.mp4   -> this folder
  /images/hero-poster.jpg -> a fallback image shown before the video loads (optional, add to ../images/)
